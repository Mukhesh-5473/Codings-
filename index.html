Patent Management System

Dynamic SQL Integration for Intellectual Property

Abstract

Efficient management and tracking of patents is crucial for inventors and organizations. This project, "Patent Management System with SQL Integration", serves as a dynamic, scalable, and user-friendly software solution for storing, retrieving, and analyzing patent data.

The transition from a volatile in-memory model to a structured MySQL database ensures persistent storage, eliminating data loss upon program termination and allowing for complex querying and data analytics. Patent records include unique IDs, titles, inventor names, filing dates, and detailed descriptions.

Application

This system acts as a foundational model for real-world intellectual property management tools. Key use cases include:

  • Educational Use: Demonstrating CRUD operations and database connectivity to students.
  • R&D Teams: Tracking innovation pipelines and internal IP catalogs.
  • Legal Teams: Managing renewal deadlines and prosecution support.
  • Government Agencies: Facilitating public patent searches and handle filings efficiently.

The cyclic workflow ensures the user can perform multiple operations in a single session, with confirmation messages displayed after every successful database transaction.

Algorithm

  1. Start program and establish MySQL connection.
  2. Display Main Menu (Search, Show All, Create, Edit, Delete, Exit).
  3. If Search: Prompt for Serial Number, Date, Status, or Region. Query database and display results.
  4. If Show All: Offer sorting (Date, Name, Status). Execute SQL and return results.
  5. If Create: Take user input, generate next serial number, and INSERT into database.
  6. If Edit: Fetch current details by S_No, update selected field, and save changes.
  7. If Delete: Confirm S_No, remove record from database upon user confirmation.
  8. If Exit: Close database connection and terminate.
