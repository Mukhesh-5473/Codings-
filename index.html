Patent Management System

SQL Integrated Patent Tracking & Analysis Tool

Python 3.8+ MySQL Database Management CRUD

Abstract

In the ever-evolving landscape of innovation and intellectual property, efficient management and tracking of patents is crucial for both inventors and organizations. This project, titled "Patent Management System with SQL Integration", is designed to serve as a dynamic, scalable, and user-friendly software solution for storing, retrieving, and analyzing patent data.

Initially implemented using in-memory storage, the system has been significantly enhanced with an SQL-based backend to ensure persistent data storage and real-time data manipulation. Users can perform keyword-based searches, filter by year or inventor, and view insightful statistics.

Application & Use Cases

This system serves as a foundational model for real-world intellectual property management tools.

  • Educational Use: Ideal for CS students learning CRUD operations and DB connectivity.
  • R&D Teams: Helps individual inventors track their innovation pipeline.
  • Legal Teams: Assists law firms in tracking patent litigation risks and renewal deadlines.
  • Government: Organizing national patent databases for public search.

System Design

The system follows a modular, cyclic workflow where every action (Search, Create, Edit, Delete) returns the user to the central Main Menu after updating the Patent Manager (MySQL Database).

[Logic Flow: Start -> Main Menu -> Action -> DB Update -> Output -> Loop]
