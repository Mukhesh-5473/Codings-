Patent Management System

Python & MySQL Integration Project

Abstract

Initially implemented using in-memory storage, this system has been significantly enhanced with an SQL-based backend to ensure persistent data storage and real-time data manipulation. The core objective is to provide a seamless interface for adding, searching, filtering, and analyzing patent records securely within a structured database.
